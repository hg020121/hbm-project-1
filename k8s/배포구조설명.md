# HBM MR-MUF AI 공정 추천 시스템 - Kubernetes 배포 구조

**제출자:** sk054 이한결  
**클래스:** SKALA 3기 2반 (class-2)  
**클러스터:** skala-edu-cluster (AWS EKS)  
**접속 URL:** http://sk054-hbm.skala25a.project.skala-ai.com

---

## 1. 서비스 개요

HBM(High Bandwidth Memory) MR-MUF 공정 데이터를 통합 관리하고, LangGraph + RAG 기반 AI 파라미터 추천을 제공하는 웹 서비스입니다.

| 구성 요소 | 역할 | 기술 스택 |
|-----------|------|-----------|
| Frontend | 공정 대시보드 UI, 엔지니어/품질 역할 전환 | Vue.js 3 + Vite + Nginx |
| API | REST API 서버, AI 추천, RAG 쿼리 처리 | FastAPI + LangGraph + FAISS |
| MySQL | 공정 데이터 통합 저장 (stacking/reflow/injection) | MySQL 8.0 |

---

## 2. 배포 아키텍처

```
[사용자 브라우저]
      │
      ▼
[Ingress (public-nginx)]
sk054-hbm.skala25a.project.skala-ai.com
      │
      ├── /api/*  ──────────────► [Service: sk054-hbm-api-svc :8000]
      │                                    │
      │                           [Deployment: sk054-hbm-api]
      │                           - FastAPI + LangGraph + FAISS
      │                           - initContainer: wait-for-mysql
      │                           - PVC: sk054-faiss-pvc (500Mi)
      │                           - envFrom: ConfigMap + Secret
      │
      └── /      ──────────────► [Service: sk054-hbm-frontend-svc :80]
                                          │
                                 [Deployment: sk054-hbm-frontend]
                                 - Vue.js + Nginx
                                 - try_files (SPA routing)

[Service: sk054-mysql-svc :3306]
        │
[Deployment: sk054-mysql]
- MySQL 8.0
- PVC: sk054-mysql-pvc (1Gi)
- envFrom: Secret (root password)
```

## 3. 적용 기술 및 리소스

### Kubernetes 리소스 목록

| 리소스 종류 | 이름 | 설명 |
|------------|------|------|
| Deployment | sk054-hbm-api | FastAPI 서버 |
| Deployment | sk054-hbm-frontend | Vue.js/Nginx 서버 |
| Deployment | sk054-mysql | MySQL 8.0 DB |
| Service | sk054-hbm-api-svc | API ClusterIP 서비스 |
| Service | sk054-hbm-frontend-svc | Frontend ClusterIP 서비스 |
| Service | sk054-mysql-svc | MySQL ClusterIP 서비스 |
| Ingress | sk054-hbm-ingress | 외부 트래픽 라우팅 |
| ConfigMap | sk054-hbm-config | 환경변수 (DB 설정, 모델명 등) |
| Secret | sk054-hbm-secret | 민감정보 (DB 패스워드, OpenAI API Key) |
| PVC | sk054-mysql-pvc | MySQL 데이터 영속성 (1Gi) |
| PVC | sk054-faiss-pvc | FAISS 벡터스토어 영속성 (500Mi) |

### 자원 할당 (LimitRange 준수)

| 서비스 | CPU Request | CPU Limit | Memory Request | Memory Limit |
|--------|-------------|-----------|----------------|--------------|
| api | 200m | 500m | 256Mi | 1Gi |
| frontend | 200m | 500m | 256Mi | 1Gi |
| mysql | 200m | 500m | 256Mi | 512Mi |

### 선택 구현 항목

- **PVC:** MySQL 데이터 및 FAISS 벡터스토어 영속성 확보
- **initContainer:** API pod가 MySQL 준비 완료 후 시작되도록 `wait-for-mysql` initContainer 적용 (`busybox:1.35`, `nc -z sk054-mysql-svc 3306`)

### Probe 설정

| 서비스 | livenessProbe | readinessProbe |
|--------|---------------|----------------|
| api | GET /health (port 8000) | GET /health (port 8000) |
| frontend | GET / (port 80) | GET / (port 80) |
| mysql | mysqladmin ping | mysqladmin ping |
